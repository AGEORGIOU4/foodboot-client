<h2>Computer Science Final Year Thesis Project</h2>
<h6>UCLan Cyprus</h6>

<p>Foodboot is a Web Platform combined with Applications running on Cloud. Its main purpose is to work as an assistive tool for nutritionists and clients.
  Nutritionists can manage and control clients' data, view important information about each client (food preferences, medical records etc), use tools such as BMI calculator,
  Water Intake Calculator, Calendar and create personalized Meal Plans for clients using a <strong>Desktop App</strong>. A <strong>Mobile App</strong> is created for the client
  that fetches all the meal plan information in an organised manner. The Web Platform is orchestrated by two web applications (Nutritionist, Client). Each application is composed
  of three main components. A cloud-operating <strong>MySQL</strong> database hosted in Heroku, the <strong>Frontend</strong> written mostly in ReactJS and the
  <strong>Backend</strong> where RESTful API architecture was used for the platform's logic and web services written in NodeJS. As a starting template for the Frontend,
  the CoreUI admin panel is used. For Authentication and user info, the platform integrates Auth0 (SaaS service). The whole project is hosted on GitHub in three different
  repositories and deployed on Netlify (links can be found below).</p>

<h4>Features of Nutritionist App</h4>
<p>
  <h6>Authentication</h6>
  <ul>
    <li>Easy on-boarding using a SaaS service</li>
    <li>Third-party social connection (Google)</li>
    <li>Branding Customization</li>
    <li>API Application to retrieve users data from Auth0</li>
  </ul>

  <h6>Profile</h6>
  <ul>
    <li>User can view profile information</li>
    <li>User can edit profile information</li>
    <li>User can delete profile</li>
  </ul>

  <h6>Client</h6>
  <ul>
    <li>CRUD operations for client</li>
    <li>Invite client to <strong>Client's App</strong> using button on clients table</li>
    <li>Search engine for clients table</li>
    <li>Sorting tables</li>
    <li>Filtering tables</li>
    <li>Import/Export clients in .csv format</li>
    <li>CRUD operations for Food Preferences of each client</li>
    <li>CRUD operations for Medical History Record of each client</li>
    <li>Print Client's 'View Client Card'</li>
  </ul>

  <h6>Calendar</h6>
  <ul>
    <li>View Calendar modes (day, week, month | list)</li>
    <li>CRUD operations for calendar events</li>
    <li>DRAG n' DROP features on Updating Events</li>
  </ul>

  <h6>Meal Plan</h6>
  <ul>
    <li>View Meal Plans as Cards Table</li>
    <li>Search engine for Meal Plans Table</li>
    <li>CRUD operations for Meal Plans</li>
    <li>Update Food Combinations per day in 'Update Meal Plan'</li>
    <li>Side Navbar that fetches and displays all client's info when Update</li>
    <li>View Personalized Meal Plan for each Client</li>
    <li>Print Personalized Meal Plan for each Client</li>
    <li>Filter Food Combinations and choices by type of meal and day</li>
    <li>Zoom in / Zoom out / Strech screen on View Meal Plan</li>
  </ul>
</p>
